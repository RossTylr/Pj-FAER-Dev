## Layer key (three-way, per the updated rubric)

- **Intrinsic** — outcome-determining. One wrong → the result is an
  artefact, not a finding.
- **Validity** — does not change the result; decides whether the result can
  be trusted. Mostly binary → treat as a GATE (pass before scoring), not a
  weighted axis.
- **Surface** — observation/presentation. Must NOT score as intrinsic; if a
  review rates one as outcome-bearing, the rubric is miscalibrated.

## The single decisive probe: parameterise, don't author

Clock 1's shape and Clock 2's steepness are the two most tempting things to
hand-draw. If both are fixed, the winner is baked in and the experiment is a
mirror, not a finding. Both MUST be swept inputs; the crossover **boundary**
is the result, the point estimate merely illustration. Tied condition: the
care-quality gradient (forward < rear) must be non-zero, or the lines never
cross.
